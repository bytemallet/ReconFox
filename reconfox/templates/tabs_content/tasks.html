Mega Tasks

A Megatask is a powerful, combined task that organizes multiple smaller tasks to work seamlessly toward a common goal, such as retrieving as many emails as possible.

Email Discovery

This task sequence automates email discovery by gathering emails from URLs, advanced search queries (dorks), and metadata from downloaded files, then identifies email patterns to streamline contact information extraction and organization.

Find Emails → Find Emails From URLs → Execute Dorks → Find Emails From Dorks → Get Files from URLs → Download All Files → Get Metadata → Get Emails From Metadata → Find Email Patterns
